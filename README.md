Tic Tac Toe : 

To start the server:
1. clone the repo
2. npm install
3. npm run start-dev

Then: 
To play with two terminals: 
-  in another terminal: node client.js
-  in another terminal: node client.js 

To play with two browsers: 
- open the index.html in a browser
- open the index.html in another broswer
